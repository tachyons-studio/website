Tachyons Studio

Create HTML webpages — fast!

Mix and match pre-made templates to create responsive, mobile-first webpages.

Getting Started

Copy the line of code below and paste it in the head of the html file(s) you want to include tachyons in.

<link rel="stylesheet" href="https://unpkg.com/tachyons@4.10.0/css/tachyons.min.css"/>

HTML template

Save this file to your computer to start prototyping right away without worrying about setting up a dev environment. You can open the file in a web browser and view your changes.

This template is always linked to the most up to date version of Tachyons.

<!DOCTYPE html>
<html lang="en">
<head>
  <title>Tachyons Studio</title>
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://unpkg.com/tachyons/css/tachyons.min.css">
  </head>
  <body>
   <-- add your sections here -->
  </body>
</html>
